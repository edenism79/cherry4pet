# 🔍 브라우저 개발자 도구 확인 방법

## 1단계: 개발자 도구 열기
- **F12** 키를 누르세요
- 또는 마우스 우클릭 → "검사" 클릭

## 2단계: Console 탭 확인
1. 상단에 있는 **Console** 탭 클릭
2. 빨간색 에러 메시지가 있나요?
3. 있다면 그 내용을 복사해주세요

## 3단계: Network 탭 확인
1. 상단에 있는 **Network** 탭 클릭
2. 페이지 새로고침 (F5)
3. `/admin/login` 요청이 계속 반복되나요?
4. 몇 개나 보이나요?

## 4단계: 스크린샷
- Console 탭 전체 화면을 캡처해주세요
- Network 탭도 캡처해주세요

이 정보를 보내주시면 정확한 원인을 찾을 수 있습니다!
